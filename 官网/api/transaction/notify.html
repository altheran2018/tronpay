# 异步回调

支付成功后，Tronpay会向目标服务器发生异步通知，告知该笔交易已经支付完成。
失败Tronpay最高最多重试5次，请注意验证消息签名。
目标服务器处理完成后请返回字符串ok即可，否则Tronpay会一直重试发送消息，最高5次

POST 【异步回调地址】

Body 请求参数

{
  "trade_id": "202308221692680418158858",
  "order_id": "202308221300188505",
  "currency": "CNY",
  "amount": 50,
  "actual_amount": 7.143,
  "token": "TGPzzSDzuWchyxEGRgBGuL2ZUm7yJMNXM8",
  "block_transaction_id": "111111111111111111",
  "signature": "xxxxxxxxxxxxxxxxxx",
  "status": 2
}

# 请求参数

名称位置类型必选中文名说明
bodybodyobject否
» trade_idbodystring是交易号
» order_idbodystring是请求支付订单号
» currencybodystring是订单币种
» amountbodyfloat是订单金额小数点保留后2位
» actual_amountbodyfloat是实际需要支付的usdt金额(USDT)小数点保留后4位
» tokenbodystring是钱包地址
» block_transaction_idbodystring是区块交易号
» signaturebodystring是签名见接口约定
» statusbodyint是订单状态1：等待支付，2：支付成功，3：已过期
