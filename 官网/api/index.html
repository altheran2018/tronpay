# 准备工作

欢迎阅读 Tronpay 技术接入文档。在开始接入之前，你需要：

  1. 访问并登录 商户后台。
  2. 在商户后台查看技术接入必需的 商户ID（Merchant_id） & 商户密钥（Secret）。
  3. 在商户后台配置你的 收款地址。
  4. 阅读我们 接口约定 及 安全说明 相关的内容。
  5. 然后你可以参考后续文档，并尝试通过接口发起 订单创建 请求。

说明

你也可以直接参考 接入工具，快速接入我们的接口服务。
